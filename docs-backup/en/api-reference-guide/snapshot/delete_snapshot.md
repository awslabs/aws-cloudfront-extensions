## delete_snapshot

- HTTP request method: `POST`

- Request

```json
/snapshot/delete_snapshot?distributionId=E20GR9AX7K798K&snapShotName=dfdfd
```

- Request body parameters
    - distributionId: String
    - snapShotName: String


- Response

```json
{
  "statusCode": 200,
  "body": "succeed delete snapshot"
}
```
