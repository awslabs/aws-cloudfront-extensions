CloudFront分配

执行如下步骤删除CloudFront分配

  1. 进入CloudFront console: https://console.aws.amazon.com/cloudfront

  2. 在CloudFront分配的搜索框中输入cloudfront-extension-workshop并搜索。然后点击右上角的禁用按钮

  3. 等待直到禁用操作完成，然后点击删除按钮
